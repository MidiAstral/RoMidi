What it can do

Everything in one script

RoMidi packs a complete MIDI playback system into a single Lua file with a polished, responsive UI built entirely with Roblox UI objects.

▶

Playback Engine

Play, pause, stop and loop any MIDI song with frame-accurate timing and smooth progress tracking.

⏲

BPM Control

Set the tempo manually or enable Auto BPM to follow the MIDI file's embedded tempo changes in real time.

📚

Song Library

Browse built-in preset categories from GitHub or load your own .mid files from workspace folders.

●

Record Mode

Hide the entire UI before recording. A 3-second countdown begins, and a triple-tap restores the window.

🔒

Mini Mode

Collapse into a compact floating controller that still lets you play, pause, loop and seek.

🔔

Toast Alerts

Animated slide-in notifications confirm every action with color-coded icons and a progress drain bar.

⚡

Anti-Lag

One-tap anti-lag script injection directly from the UI to keep gameplay smooth during playback.

🎵

MIDI Parser

Full multi-track MIDI parser with running status, tempo events, and stuck-note prevention built in.

▶ Playback

Play, Pause, Stop and Loop

Three large icon buttons give instant control over playback. Each button uses tinted icon images with a text fallback in case the image has not loaded yet.

  • PLAY starts the song; tap again while playing to pause; tap once more to resume
  • STOP halts playback and resets the position back to the beginning
  • LOOP toggles repeat mode, the song restarts after 1 second when it finishes
  • Button colors change state: green for play, rose for stop, amber for loop
  • Ripple animation on every button press for tactile feedback

⏱ BPM

Manual and Auto BPM

Type any tempo between 1 and 999 BPM, or flip on Auto BPM and let the MIDI file drive the tempo automatically including mid-song changes.

  • Auto BPM reads embedded tempo markers from the MIDI data
  • BPM updates instantly while playing, with no interruption to the song
  • Switching Auto off restores the last manually entered BPM
  • The input field locks automatically when Auto BPM is enabled

Press the LIST button in the header to open the Song Library panel. It slides in alongside the main window and stays open until manually closed.

□ Mini Mode

Compact Mini Controller

Tap MINI in the header to collapse the entire interface into a tiny floating bar called RoMini. It keeps all playback functions available while taking up minimal screen space.

● Record Mode

Hands-Free Record Mode

Toggle REC in the utility row or Mini bar to arm record mode. When you press PLAY, the entire UI disappears after a 3-second countdown so nothing blocks your screen.

  • REC button glows red while armed; status changes to "REC Ready"
  • A visible 3-2-1 countdown toast appears before the UI hides
  • Triple-tap anywhere on the screen to bring the UI back mid-song
  • Pressing STOP automatically restores the full interface
  • Works in both Main and Mini modes

📁 Custom MIDI

Load Your Own MIDI Files

Drop any .mid or .smf file into a folder inside Roblox Studio's Workspace. RoMidi automatically discovers folders and turns each one into a browseable category.

  • Each subfolder in Workspace becomes its own category tab
  • Press RELOAD in the utility row to pick up newly added files
  • Switch between local and preset songs at any time
  • The MIDI parser handles multi-track files, running status, and tempo events
  • Orphaned note-off events are cleaned automatically to prevent stuck notes

Quick Guide

Getting Started

From zero to playing your first song in under a minute.

1

Run the Script

Execute the Lua file in your Roblox executor. A loading screen appears while preset songs and images load in the background.

2

Open the Song Library

Click the LIST button in the top-right of the main panel. Choose Preset for built-in songs or Local to browse MIDI files from your workspace folders.

3

Pick a Song

Use the search bar to filter, then tap any song row to load it. A green toast confirms the song is ready and the title appears in the main panel.

4

Set Your BPM (Optional)

Leave Auto: ON to follow the MIDI's embedded tempo, or turn it off and type your own BPM value into the input box.

5

Press Play

Hit the green PLAY button. Watch the progress bar advance and the BPM counter update in real time. Tap PLAY again to pause, or STOP to reset.

6

Record (Optional)

Enable REC before pressing PLAY. The UI will count down from 3 and hide itself. Triple-tap the screen to restore it, or press STOP when finished.
