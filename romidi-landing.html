RoMidi - Help
PLAY

Play - Pause - Resume

One button, three states. Starts playback if idle. Pauses if playing. Resumes from the exact position if paused. The status bar updates live between "Playing" and "Paused."

STOP

Stop and Reset

Immediately stops playback, releases all held keyboard keys to prevent stuck notes, and resets the playhead back to the beginning.

LOOP

Loop Mode

The song automatically restarts the moment the last note finishes, with no gap. Button glows green when active.

BPM

BPM and Auto Tempo

Manually enter 1-999 to speed up or slow down playback. Enable Auto to let RoMidi follow the Set Tempo (0x51) markers embedded in the MIDI file - the BPM updates automatically in real time as the song progresses through different tempo sections.

REC

Record Mode

Hides the entire RoMidi UI before playback. On Play, a 3-second silent countdown runs, then notes begin - leaving your recording clean. Triple-tap the screen to restore the UI mid-recording.

LAG

Anti-Lag

Fetches and runs an external Lua optimization script to clean up memory and reduce frame drops on busy Roblox servers. Shows "OK" on success, "ERR" if the request fails.

RELOAD

Reload Library

Clears all cached MIDI data and rescans your workspace folders live. Add new songs without rejoining the server.

What is RoMidi

Your automated piano performer

A Lua executor script that reads standard MIDI files and performs them live inside Roblox piano games - no manual key presses needed.

How does it work?

RoMidi reads .mid files from folders you place in your Roblox workspace. It parses every MIDI event - note-on, note-off, tempo changes, chords - and builds a precise timeline of keys to press and release.

Playback runs in its own coroutine with frame-level timing, firing real keyboard inputs through Roblox's VirtualInputManager. The piano in your game responds exactly as if you were playing by hand.

Because only standard keyboard inputs are used, RoMidi works with virtually any Roblox piano game that reads keyboard input, including NeoPiano.

Enable REC Mode

Press REC. The button turns bright red and the status shows "REC Ready." The UI is still visible until you press Play.

2

Press Play - 3-second countdown

Every panel disappears instantly. RoMidi silently counts 3 seconds before the first note fires, so your recording never starts mid-phrase.

3

Piano plays - screen is clean

Your recording captures only the Roblox piano. RoMidi runs invisibly in the background, pressing keys at exact timing.

4

Triple-tap to restore

Tap or click 3 times within 1.5 seconds at any point and the UI fades back in with a spring animation - without stopping or interrupting playback.
